# Pixel Painter on C# - форк отличного проекта, который я захотел сделать доступнее для большинства людей

# 🇬🇧 English

Convert any image into a pixel-by-pixel mouse painting — **completely automatically!**

## 🚀 How it works
1. **Load** an image into the app.
2. **Select the area** on your screen where you want to paint.
3. **Map the colors** (show the bot where the color palette is located).
4. **Adjust the delay** between clicks to match your performance.
5. Press **Start** to begin, and press `ESC` to **stop** at any time.

## 📦 Installation
1. Go to the **Releases** section on the right.
2. Download the latest installer (`.exe`).
3. Run it and you are ready to go!

## 🎮 Supported Software
You can use it not only in standard MS Paint but also in various applications and games with built-in painters, such as **Tomodachi Life: Living The Dream** and many others!

# 🇷🇺 Русский

Превращай любое изображение в пиксельный рисунок мышкой — **полностью автоматически!**

## 🚀 Как это работает
1. **Загрузи** изображение в приложение.
2. **Выбери область** на экране, где ты хочешь рисовать.
3. **Настрой палитру** (покажи боту, где находятся цвета в нужной программе).
4. **Отрегулируй задержку** между кликами для оптимальной скорости.
5. Нажми **Start** для запуска и `ESC` для **мгновенной остановки** в любой момент.

## 📦 Установка
1. Перейди в раздел **Releases** (Релизы) справа.
2. Скачай самый свежий установщик (`.exe` файл).
3. Запусти его — и всё готово к работе!

## 🎮 Где можно использовать?
Утилита отлично работает не только в обычном MS Paint, но и в любых других приложениях и играх со встроенными графическими редакторами, например в **Tomodachi Life: Living The Dream** и аналогичных проектах!
